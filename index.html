Quick Start

# Clone the repository
git clone https://github.com/JoshuaAFerguson/terminal-velocity.git
cd terminal-velocity

# Quick start with Docker
cp .env.example .env
# Edit .env and set DB_PASSWORD
docker compose up -d

# Connect to the game
ssh -p 2222 username@localhost
